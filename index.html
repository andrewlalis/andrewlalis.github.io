About Andrew

Andrew is an experienced back-end software engineer that focuses on web systems and persistence mechanisms. He has worked with low-level embedded microcontrollers, all the way up to containerized cloud services and distributed databases. In his free time, he likes to work on hobby programming projects mostly in Java and D, some of which you can check out on the Projects page. Professionally, he currently offers software consulting and contracting services to individuals and small to medium-sized businesses via Gyrobian.

More specifically, he has experience in the following technical categories:

  • More than 3 years in back-end PHP web development.
  • More than 6 years of experience in Java for web API and service development, including Spring Boot.
  • Oracle Certified Java SE 11 Professional (and Java SE 7 Programmer)
  • More than 6 years of experience with various SQL databases, such as Postgres, MySQL, MariaDB.
  • More than 5 years experience as a technical supervisor for multiple university level courses, involving workflow automation for hundreds of students.
  • More than 3 years of experience in Linux system administration, including SystemD, Bash scripting, Cron, and deployment configurations.
  • Over 2 years of experience in the following languages: Python, Lua, C, HTML/CSS, Javascript

Andrew also sometimes works to find solutions to technical Minecraft problems. These range from basic redstone innovations, to complex systems rooted in the fundamentals of computing science. Below you can see a short listing of some of the things he's done.

SpeedCarts

Created a mod to increase minecart speeds, in order to make minecart travel a viable alternative to other options like the Elytra or boats.

RailSignalAPI

Built with Spring Boot and using websockets and/or plain TCP sockets, RailSignalAPI is able to interface with nearly any ingame system to build working signal systems that update in real-time. Also provides a live map for users to see updates.

Watch a demonstration on YouTube, or check out the code on GitHub.

Intelligent Minecart Routing

A design and proof-of-concept for a distributed, stack-based minecart routing system that enables automatic routing through rail networks that are arranged as directed graphs.
